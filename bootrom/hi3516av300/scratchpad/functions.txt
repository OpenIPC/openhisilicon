hi3516av300 bootrom — function ledger
======================================

Format: <addr> <name> <signature/notes>

Reversed in re/bootloader.c + re/start.S:

func_00  0x040000e0  memcpy32(src, dst, size)               LDM/STM r3-r10 loop, used by _start to copy clear_remap thunk to SDRAM
func_01  0x04002844  disable_ctrl_features                  MRC/BIC c1c0: clear I-cache, D-cache, MMU, branch prediction, align check
func_02  0x04002858  enable_instruction_cache               sets parity, calls invalidate_instruction_pipeline, enables I-cache, ISB
func_03  0x04002880  disable_instruction_cache              clears I-cache enable, tail-calls invalidate_instruction_pipeline
func_04  0x040028e0  invalidate_instruction_pipeline        MCR p15 c8c7 (TLB) + MCR p15 c7c5 (I-cache)
func_05  0x04002900  memset(dst, val, size)                 unrolled 8-byte STM with byte-tail
func_06  0x040029a0  memcpy(dst, src, size)                 aligned + unaligned paths, unrolled
func_07  0x04002cd8  strlen(str)                            LDRB + CMP, returns end-start
func_08  0x04002d04  strncmp(s1, s2, n)                     two-pointer compare, returns signed diff
func_09  0x04002e00  strcmp(s1, s2)                         two-pointer compare until first mismatch or NUL
func_0a  0x04000140  configure_soc_clocks                   polls PLL_LOCK_STAT for APLL+DPLL+EPLL (mask 0xb, timeout 5001 iters), writes 0x592 into SOC_CONFIG, NOP-settles 10 cycles
func_0b  0x040001b0  reset_soc_clocks                       inverse of configure_soc_clocks — clears SOC_CONFIG bits 0-10 (bit 9 preserved), 10-NOP settle
func_0c  0x04002704  main                                   bootloader main flow — 4 boot paths dispatched on SYSSTAT bits 4/5 + PERISTAT bit 0; converges on success (disable_ic + maybe reset_soc_clocks + timer_stop) or erase+SYSRES soft-reset
func_0d  0x0400105c  is_update_mode                         polls GPIO0 pin 0 — recovery button held LOW for ~20s (two timer_wait_seconds(10) gaps) returns 1
func_0e  0x04001128  initial_memory_setup                   enable I-cache, copy 12-byte record from bootrom 0x7f44 to SRAM 0x04010004, then organize_memory_chunks
func_0f  0x04002284  bootrom_secure_check                   ungate KLAD/RSA/TRNG/SPACC via CRG[0x1a0], kick TRNG, dispatch among klad_{load_keys,verify,alt_verify,unlock} based on SYSSTAT bits 4/5 + recovery button
func_10  0x04002368  fastboot_update                        burn_bootloader over UART; on success consult SYSSTAT.bit4 for eMMC reset path
func_11  0x040023c4  media_update                           PERISTAT bits 4/5 dispatch between fastboot-media (init_a/b/c + program_a + finalize_a + "ziju" magic) and alt path (init_alt + set_boot_target_addr)
func_12  0x0400249c  config_use_flash                       flash boot — memcpy 0x6000 from FLASH_START to SRAM, parse header (magic "MBSB"), relocate second-stage into DDR+0x01000000, four-word cross-check
func_13  0x040025d4  bootrom_alt_media                      eMMC alt boot — initialize_emmc + memcpy_emmc into SRAM, parse same "MBSB" header, relocate into DDR, four-word cross-check
func_14  0x040051bc  timer_start                            ungate timer clock via SC_CTRL bit 16, TIMER0_CONTROL = 0xc2
func_15  0x040051e8  timer_stop                             TIMER0_CONTROL = 0
func_16  0x040051f8  timer_reset_counter                    LOAD = 0xFFFFFFFF, restart timer, snapshot value into SRAM[0x180]
func_17  0x04002cb4  timer_wait_seconds(n)                  loops n × wait_timer_0(1000); returns immediately if n=0

Reset trampoline (covered by re/start.S, not in the func_N enumeration):

  0x04000000  _entry                  B _start
  0x04000004  exception vectors       all branch to 0x00000100 (post-remap SDRAM)
  0x0400003c  _magic                  .word 0x12345678
  0x04000040  clear_remap_rom_ptr     0x0400009c
  0x04000044  clear_remap_ptr         0x04016500
  0x04000048  _start                  CPSR setup → disable_ctrl_features →
                                      stack → configure_soc_clocks →
                                      zero SDRAM[0..0x268] → main → copy
                                      clear_remap thunk to SDRAM → jump
  0x0400009c  clear_remap thunk       sets SC_CTRL.bit8 to release boot
                                      remap, jumps to *(SDRAM_START)
  0x04000100  _dead_loop              triggers SC_SYSRES soft-reset, spins
